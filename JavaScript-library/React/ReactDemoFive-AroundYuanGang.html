突然想起一个词，叫：食在广州。

而我，广漂，衣食住行都围绕广州番禺区的【员岗地铁站】。

所以，兴致而来，开发一个多端登录的 App，记录附近的点点滴滴。

3.2 食

  1. 对一些饭点的介绍
  2. 点击地图的饭点，可以查看到对应饭点的菜单（建议堂食、不建议堂食）
  3. 可以获取饭点的外卖电话、点餐 App（美团，大众，饿了么……）
  4. 可以评价饭点的饭菜
  5. 店家可以更新自己的菜单（收费）
